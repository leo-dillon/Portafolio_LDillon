Desafíos de Programación

Como desarrollador me gusta realizar desafíos técnicos. Estos desafíos me permiten mejorar mis habilidades de lógica, pensamiento critico y creatividad.

La mayoría de los desafíos que resuelvo provienen de la plataforma LeetCode, un sitio muy completo que reúne problemas de distintas temáticas y niveles de dificultad.

Cada desafío que completo lo documento dentro de este proyecto, organizándolo por nivel de dificultad (Fácil, Medio, o Difícil) y acompañándolo con un enlace directo al desafío en LeetCode.

Mi objetivo con este repositorio es compartir mi forma de resolver los distintos desafíos y dejar registro de los avances que voy logrando.

Tecnologías utilizadas en el proyecto :

html
Vue
Tailwind
JavaScript

Home de Desafíos

Este es el punto de partida del proyecto Desafíos de Programación. Aquí encontrarás todos los retos que voy resolviendo, organizados en tres niveles de dificultad: Fácil, Medio y Difícil. Podrás conocer cuántos desafíos he completado, filtrarlos por nivel y explorarlos en el orden en que fueron realizados.

Modal del Ejercicio resuelto

Al abrir un desafío, se mostrará un modal con el título, un enlace directo a LeetCode, la descripción del reto y algunos ejemplos de las variables requeridas. También encontrarás mi solución paso a paso y un área interactiva donde podrás ejecutar el código para ver cómo funciona.
